## Domain Description

Inference Router simulates a distributed large language model inference system that receives user prompts and routes them across multiple mock server models. A single server can become stagnant and slow when a lot users submit requests at the same time because LLM inferencing requires significant computation, memory, and processing time due the kv cache and request's prompt size, which can cause long queues, slow responses, or possibly system failure. The system addresses this scalability problem by distributing requests across multiple workers based on their current workload and availability. This domain is meaningful from a Computing for the Common Good perspective because reliable and accessible AI services can support users who depend on language technologies for information and assistance. When the system works well, users receive timely response, but when it becomes overloaded or unavailable,  users, who depend on this for work, school, or urgent advice, can be negatively affected the most.
